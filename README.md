# Character Counter

A simple and responsive **Character Counter** web application built using **HTML, CSS, and JavaScript**. It allows users to type text in a textarea while displaying the number of characters entered, the remaining characters available, and a warning when the maximum limit is reached.

## 🚀 Features

* Real-time character counting
* Displays remaining characters
* Maximum limit of **200 characters**
* Warning message when the character limit is reached
* Responsive and modern user interface
* Clean and beginner-friendly code

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

## ▶️ How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Double-click **index.html** or open it in your preferred web browser.

No installation or additional libraries are required.

## 💡 How It Works

* Users type inside the textarea.
* JavaScript listens for every input event.
* The application updates:

  * Current character count
  * Remaining characters
  * Warning message when the 200-character limit is reached
